Assignment 4: (60 Mins)
Write the program to display students' grades based on input score. The criteria for
display grades are below.
Score 0-49 : Grade F
Score 50-59: Grade E
Score 60-69: Grade D
Score 70-79: Grade C
Score 80-89: Grade B
Score 90-100: Grade A
